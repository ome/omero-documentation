*************
Using Scripts
*************

OMERO.insight allows you to use scripts to add functionality to the programme. OMERO.insight comes with some built in scripts which enable output of data in a number of formats, as well as other functionality.

1. Click on the **Run Scripts** icon in the toolbar .

.. image:: /images/tutor/scriptsMenu.png
    :width: 80%
    :align: right

2. To create a figure comprising thumbnails, either select the images for inclusion in the data tree or thumbnails in the centre pane, or select one or more Datasets.
Select **Figure Scripts > Thumbnail Figure**

.. image:: /images/tutor/scriptsThumbnails.png
    :width: 85%
    :align: right

3. The script dialogue shows the IDs of the selected images (or ID of the Dataset/s if  they are selected).

.. image:: /images/tutor/scriptsDefault.png
    :width: 100%
    :align: center

4. Set the minimum and maximum thumbnail size (defaults are 100).
Set the maximum number of columns (default 10).

.. image:: /images/tutor/scriptsSize.png
    :width: 75%
    :align: center

5. Select the format for the figure (default JPEG).

.. image:: /images/tutor/scriptsFormat.png
    :width: 85%
    :align: center

6. Click on the **Script** menu to download or view the script.
Click **Run Script** to finish.

.. image:: /images/tutor/scriptsRun.png
    :width: 100%
    :align: center

9. Click on the image or dataset the figure has been attached to.
In the right pane, click on drop-down icon after the attachment to view or download.

.. image:: /images/tutor/scriptsView.png
    :width: 100%
    :align: center
