*****************************************
Getting started with OMERO.insight client
*****************************************
OMERO stores image data on a central server. You can use the OMERO.insight client to upload, view and download data from any personal computer.
Your institution's OMERO server administrator will provide you with the server address to use when connecting from OMERO.insight. They may have set up a user name and password for you too. Alternatively your username and password may be your standard login for the institution's networked computers.

Installing
==========
#. Download OMERO.insight client.
   `OMERO downloads page <https://www.openmicroscopy.org/site/support/omero4/downloads>`_

   .. image:: /images/tutor/gettingStartedDownloads.png
       :width: 100%
       :align: center

   |

#. Expand OMERO.insight client archive.

   .. image:: /images/tutor/gettingStartedZips.png
       :width: 80%
       :align: center

   |

#. Open OMERO.insight Client.

   .. image:: /images/tutor/gettingStartedApps.png
       :width: 100%
       :align: center

   |

Adding server and logging in
============================
Adding the server is only required the first time an OMERO server is accessed from the computer.

1. Click on the **Spanner** icon.

   .. image:: /images/tutor/gettingStartedSpanner.png
       :width: 75%
       :align: center

|

2. In the **Add Server** dialogue box, click on the **+** icon.

   .. image:: /images/tutor/gettingStartedAdd.png
       :width: 75%
       :align: center

|

3. Enter the server address provided by your OMERO administrator.
   Click on **Apply**.

   .. image:: /images/tutor/gettingStartedApply.png
       :width: 75%
       :align: center

|

4. Enter the **Username** and **Password** provided by your OMERO administrator.
   Click on **Login**.

   .. image:: /images/tutor/gettingStartedLogin.png
       :width: 75%
       :align: center

|

Adding image data
=================
1. Click on the **Importer** icon.

   .. image:: /images/tutor/gettingStartedDataManager.png
       :width: 88%
       :align: center

|

2. Add image data using the **File Chooser**.

   .. image:: /images/tutor/gettingStartedFileChooser.png
       :width: 88%
       :align: center

|

3. Select image data to be imported.
   Click the right **Add** arrow.

   .. image:: /images/tutor/gettingStartedData.png
       :width: 100%
       :align: center

|

4. Specify location for saving imported data:

   - a new project and/or dataset can be created

   - an existing project and/or dataset can be selected

   - a new dataset can be automatically created from the source folder.

   .. image:: /images/tutor/gettingStartedProject.png
       :width: 100%
       :align: center

|

5. Click **Add to the Queue**.

   .. image:: /images/tutor/gettingStartedQueue.png
       :width: 75%
       :align: center

|

6. Data to be imported appears in queue pane.
   A copy of the original data can be archived in its original format by checking the box in the right most column.

   .. image:: /images/tutor/gettingStartedArchive.png
       :width: 100%
       :align: center

|

7. Click on **Import**.

   .. image:: /images/tutor/gettingStartedImport.png
       :width: 75%
       :align: center

|

8. Import progress will be shown in the **Import #** 1 tab.
   Further data for import can be queued and imported using the Select Data for Import tab.
   Multiple queues can be run simultaneously.

   .. image:: /images/tutor/gettingStartedComplete.png
       :width: 100%
       :align: center

|

9. Hovering over a thumbnail of the imported image shows a larger preview.
   Double clicking on the thumbnail opens it in the image viewer window.
   Clicking on the hyperlink takes you to the data manager window and dataset into which the data was imported.

   .. image:: /images/tutor/gettingStartedClose.png
       :width: 100%
       :align: center

|

10. The imported data can be seen in the data manager.

   .. image:: /images/tutor/gettingStartedImportedData.png
       :width: 75%
       :align: center

|

Viewing image data
==================
1. The data manager window is divided into 3 panes:

   - the left hand pane shows the data tree and tabs for screens, attachments, tags and search

   - the centre pane shows thumbnails and enables filtering of searches

   - the right hand pane shows he metadata for the data and a preview.

   .. image:: /images/tutor/gettingStartedPanes.png
       :width: 75%
       :align: center

|

2. Double click the image name or thumbnail to open an image in the full viewer or click on the image viewer icon.

   .. image:: /images/tutor/gettingStartedPanes.png
       :width: 75%
       :align: center

|

3. The image viewer enables you to change rendering settings.
   Hide / show rendering settings pane.
   Toggle between colour and monochrome rendering.
   Toggle the **Advanced** tab on the right of the rendering settings pane to show more controls.

   .. image:: /images/tutor/gettingStartedRendering.png
       :width: 100%
       :align: center

|

4. Click the colour picker to open the colour picker window.
   This is used to change the colour rendering for a channel.
   Clicking the colour well toggles the channel on or off

   .. image:: /images/tutor/gettingStartedPicker.png
       :width: 100%
       :align: center

|

5. Click on **Metadata** button to view the metadata in the right pane.

   .. image:: /images/tutor/gettingStartedMetadata.png
       :width: 100%
       :align: center

|

6. Click on Measurement Tool button to open the Measurement Tools palette.

   .. image:: /images/tutor/gettingStartedMetadata.png
       :width: 90%
       :align: center

|

7. Use the toolbar to draw Regions of Interest (ROIs).
   Click on the tabs to see details of ROIs or view results of analysis based on them.

   .. image:: /images/tutor/gettingStartedTools.png
       :width: 100%
       :align: center

|

8. Use the sliders to step through **Z-stacks** and **T-sequences** or click the **Play** triangle to play through them as a sequence.

   .. image:: /images/tutor/gettingStartedStacks.png
       :width: 100%
       :align: center

|

Using Web Start
===============
If users have restrictions on their ability to install software on their computers, a version of the OMERO.insight client can be downloaded as a Web Start package via the OMERO.web client.
The Web Start download has to be repeated each session, but this has the advantage of ensuring an up-to-date version is always used. For all practical purposes the Web Start OMERO.insight client is identical to the standard OMERO.insight client.
To access the OMERO.web client you will need a web address (URL) provided by your OMERO administrator. Your username and password to log in is the same used for the OMERO.insight client.

1. Click on the **Web Start** button.

   .. image:: /images/tutor/gettingStartedStacks.png
       :width: 100%
       :align: center

|

2. On Windows the Java package usually downloads and starts unzipping automatically.
   On the Mac you may have to double-click the downloaded file to expand it.

   .. image:: /images/tutor/gettingStartedDownload.png
       :width: 100%
       :align: center

|

3. On Windows click **Run** to open OMERO.insight.
   On a Mac click **Allow** x.
   The saved application is then used like any other programme.
   Web Start OMERO.insight will have to be downloaded each session, but this ensures you automatically have the latest OMERO.insight version.

   .. image:: /images/tutor/gettingStartedStacks.png
       :width: 100%
       :align: center
