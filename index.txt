#############################
OMERO |release| Documentation
#############################

This documentation is a work in progress and many aspects of OMERO are
not yet covered. The first three sections of this documentation cover the basic usage of OMERO as well as training material suitable for most end-users. System administrators wanting to install an OMERO server will be more interested by the :ref:`serverinstallation` and subsequent sections. The first part of this document covers the user documentation. Finally, developers can find more specific information about OMERO in the :ref:`developerdocumentation`.
