camel casing rules already in xsd-fu. Some will need to be adjusted to
match or special case code added to xsd-fu. These are (XMLAnnotation,
CO2Percentage, PinholeSize?), and these will be changed to match the OME
Data Model version. See: `#8041 </ome/ticket/8041>`_ Entries requiring
no work removed form the list.
